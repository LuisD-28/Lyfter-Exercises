Ejercicio 1.

   ¿Cuál es la complejidad de cada versión?
    -Version 1: O(n)
    -Version 2: O(1)
   
   ¿Qué versión usaría si number = 1 000 000 000? ¿Por qué?
    Para un numero tan grande la mejor opcion es la version 2 ya que la formula no depende del tamaño del numero caso contrario a la version 1 que cosumiria demaciado tiempo en iteraciones.

Ejercicio 2.

   ¿Cuál es la complejidad de cada Algoritmo?
    -Linear Search: O(1)
    -Binary Search: O(1)

   ¿En qué condiciones conviene usar cada uno?
    Cuando la lista es pequeña conviene usar Linear Search.
    Cuando se realizaran multiples busquedas o la lista es muy grande conviene usar Binary Search.
    
   ¿Qué pasa si la lista no está ordenada?
    Binary Search no funcionaria por que el algoritmo depende del orden para descartar la mitad de la lista, la function toma el elemento del medio y dependiendo si es mayor o menor relizara
    la busqueda a la izquierda o derecha pero en una lista no ordenada no tiene sentido.
